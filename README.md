## 模仿Webstack风格的导航
* 无需VPS
* 无需数据库
* 无需懂代码

## 只需修改三处
* 全局站点配置：修改根目录下的config.toml文件
* 图片（logo等)：替换/docs/assets/images下的同名文件
* 站点内容修改：修改/data/目录下的yml文件
